## what this is

DRBL Exchange is a distributed resource building lab — a peer-run mutual aid space for the decentralised tech ecosystem. It exists to create access for people to find funding ethically, connect on projects, and build shared infrastructure — without extraction, without hierarchy, without the usual nonsense.

Entry is by introduction. Someone who's already here vouches for you. That's the whole filter — and it's intentional.

---

## how it works

DRBL Exchange currently lives on Discord while **Moss** — a decentralised collaboration hub for social weaving — is in development. Discord is the interim. Moss is the destination. Both are invite-only.

### the channels

| channel | purpose |
|---------|---------|
| `#connect` | Introduce yourself — who you are, what you're working on, what you offer, what you need |
| `#needs` | One need per message. What it is, who it helps, what done looks like, deadline and timezone |
| `#offers` | What you can do, who it's for, your availability and limits |
| `#matches` | Need meets offer. Name an owner. Name a next step. 48-hour moves. 2-week sprints. |

### the rhythm
- 48-hour response moves
- 2-week sprints
- Ship over debate

## the tenets

DRBL Exchange runs on mutuality. Honesty, consistency, repair, and respect for boundaries.

1. **Non-extractive** — clear asks, offer something back or timebox. This is an exchange, not a favour bank.
2. **Consent first** — no unsolicited DMs. Ask in-channel before sliding into anyone's inbox.
3. **One item per post** — one need, one offer, one idea. Clarity is kindness.
4. **Ship over debate** — propose, name an owner, set a deadline. Progress beats perfect.
5. **Credit + permission** — don't repost others' work outside drblx without asking.
6. **No harassment or pile-ons** — critique ideas, not people. Mods enforce this without debate.
7. **Keep it public-safe** — treat every post as if anyone could read it.
8. **No epistemic bullying** — debate-as-domination is not welcome and will result in removal.
